# Loan Prediction System

This is a web application that predicts loan approval using a machine learning model.

## Features
- Predicts whether a loan will be approved based on user input.
- Uses Logistic Regression model.
- Frontend built with HTML/CSS and Flask.
- Interactive form for inputting applicant details.

## How to Run Locally
1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
4. Run the app:
    ```
    python app.py
    ```
5. Open browser at `http://127.0.0.1:5000`.

## Deployment
The app can be deployed on platforms like Render for online access.
